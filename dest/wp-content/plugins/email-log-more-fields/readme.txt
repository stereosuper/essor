== Description ==

An addon to Email Log Plugin, that allows you to see more fields in the email log.

This plugin is an addon to the [Email Log](https://wpemaillog.com) WordPress Plugin. You need to have the Email Log Plugin installed and activated for this plugin to work.

== Installation ==

#### Normal WordPress installations

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
